Security Considerations

The first step that should be taken in the splitting of the web application into frontend and backend components, with the admin page being accesible only through the backend. Secondly, the information from the sign up form is sent not only to the database, but also in the URL of the Thank you page. This could lead to a potential security breach. To be in compliance with the 2018 Data protection act, the customers' private information should be encrypted in the database. The application does not feature accounts and user login so password encryption is not necesarry.
Furthermore, HTTPS compliance must be guaranteed by obtaining an SSL certificate. HTTPS offers data encryption and security, offering all customers peace of mind when accessing the Chollerton Tearooms website.
